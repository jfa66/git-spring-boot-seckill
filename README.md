# 工程简介
简易秒杀系统。主要用来学习各种锁的应用
# 开发环境
Idea、Maven、jdk1.8、Springboot、Mybatis、Mysql、Redis、Zookeeper
# 启动说明
直接运行GitSpringBootSeckillApplication启动即可
